Mug Life Coffee & Bakery Website
A responsive, modern coffee shop website built with Bootstrap 5 and custom CSS.

Features
Responsive Design: Mobile-first approach with Bootstrap 5 grid system

Interactive Navigation: Smooth scroll navigation with active state

Complete Menu Display: Organized menu with coffee, tea, food, and dessert sections

Special Offers Section: Highlighted daily specials and promotions

Events & Gallery: Showcases cafe events and ambiance

Contact Form: Functional contact form with validation

Customer Testimonials: Reviews from satisfied customers

Newsletter Subscription: Email subscription for promotions

Setup Instructions
Clone or Download the project files to your local machine

Open index.html in your web browser to view the website

No build process required - all dependencies are loaded via CDN

Dependencies
Bootstrap 5.3.2: CSS framework for responsive design

Font Awesome 6.4.2: Icon library

Google Fonts: Typography (loaded via CSS)

Custom CSS Features
The website includes custom styling for:

Navigation bar with fixed positioning

Hero section with background image

Menu item cards with hover effects

Special offer badges

Gallery with overlay effects

Testimonial cards

Contact form styling

Responsive typography and spacing

Browser Compatibility
Chrome (latest)

Firefox (latest)

Safari (latest)

Edge (latest)

Image Requirements
All images should be placed in the /img directory. Ensure images are:

Optimized for web (compressed)

Properly named (no spaces, use hyphens or underscores)

In appropriate formats (JPG, PNG, WebP, AVIF)

Customization
Colors: Modify CSS variables in style.css for brand colors

Content: Update text content directly in index.html

Images: Replace images in /img directory with your own

Contact Information: Update address, phone, and email in contact section
